Materialize: OptionBox Plugin

This plugin is an attempt to create an insanely light-weight user interface control that is not quite as intrusive as a modal state dialog, but allows more than just navigation functionality offered by most drop-down components. This plugin uses the Materialize user interface library.

  • Visit the Materialize homepage
  • Download Materialize source

Specifically, the people option box on this page contains an input element. You'll notice setting focus to the element does not hide the control. OptionBox allows fully interactive content inside its container.

Future Improvements

Right now the control only detects horizontal document collision. It would be ideal to have vertical collision as well where the OptionBox could be positioned above or to the side of trigger elements.

It would be great to able to add classes to OptionBox child elements that would trigger the box to close.
